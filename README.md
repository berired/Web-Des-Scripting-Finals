# Web Des and Scripting Finals

## Project Description
This project is my final output for Web Design and Scripting. It is a responsive single-page website titled **David's Dream Garage**, focused on showcasing Porsche cars through interactive sections, animations, and media.

The site includes:
- A hero section with Porsche performance highlights
- A featured cars slider
- A filterable garage section
- Scroll-based animations and navigation behaviors
- Audio interaction for the start button

## Tech Stack
- **HTML5** for page structure and content
- **CSS3** for layout, styling, and responsiveness
- **JavaScript (Vanilla JS)** for interactivity and UI behavior
- **Swiper.js** for the featured cars carousel/slider
- **MixItUp** for filtering featured garage items
- **ScrollReveal** for scroll-based reveal animations
- **Remix Icon** for iconography
- **Google Fonts (Exo)** for typography
